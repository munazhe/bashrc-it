# bashrc-it

## Dependecies:
* rsync >= 3.2.3
* sshfs >= 3.7.1
* fuse >= 3.10.1



## Install

### For linux
1. `git clone https://github.com/munazhe/bashrc-it`
2. `cd bashrc-it`
3. `bash install`
